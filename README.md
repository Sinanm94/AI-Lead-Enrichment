# AI Lead Enrichment System
Local LLM (Llama 3.2) for real estate lead intelligence

## Results
- 20 leads analyzed
- 7.3/10 avg quality score  
- 65% high-quality leads
- **£0 cost** (vs £120/year OpenAI)

## Tech
- Llama 3.2 (Ollama)
- Python
- HTML + Chart.js

## Quick Start
```bash
python3 analyze.py
xdg-open dashboard.html
```
